h1. 2012 High Resolution Vegetation Imagery

h2. This dataset is available as a WMS.  

WMS Endpoint: "ogc.gina.alaska.edu/aea/suwa.private/veg_map_sat":http://ogc.gina.alaska.edu/aea/suwa.private/veg_map_sat 

This is license restricted data and not available for public distribution.

Please contact mitch@gina.alaska.edu if you are interested in a copy via hard disk or require wms access information.

Quick overview of the data:

* 2012 satellite mosaic
* 8bit orthorectified pan-sharpened color infrared and true color imagery
* .5 m resolution
* Terrain correction completed with Matsu IFSAR
* available under i-cubed license
